Cheat sheet, baby

Tables
code name description
<table> table Use this to contain all information for your table.
<thead> table header This is where header information goes. you can style it seperatly, and select it specifically with the code thead th:nth-child(1)
<tbody> table body Contains the main data of your table. between the table header and the table footer

Links
code name description
<img src=" "> image source Use src = to link the relative path to the image, or link to a url.
<href> hyperlink Stands for Hypertext REFerence. you can link to another tag
<a> anchor Usually seen with a href attribute to link to pages, files, addresses, locations on the same page, - anything url.

Elements
code name description
<header> header Good for intro content or set of nav links.
<button> button You guessed it: there's a button!
<form> form Where to store all your grouped input elements (creating a form for user input.)
<input> input Display varies based on "type" attribute.

The Box Model
code name description
margin margin determines the space between content and stuff around it. Remember, shorthand is a clockwise square.
margin: auto auto margin will even itself out horizontally, and YOU don't have to do the math.
visibility: hidden visibility the user can't see it, but elements must arrange themsleves around it. keeps organization and structure steady.
